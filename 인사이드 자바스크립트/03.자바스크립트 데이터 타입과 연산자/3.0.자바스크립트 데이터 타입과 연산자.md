### 기본 데이터 타입

- 자바스크립트는 크게 기본 타입과 참조 타입으로 나뉜다.

```
+-- 자바스크립트 데이터 타입
      +-- 기본 타입
            +-- 숫자(number)
            +-- 문자열(string)
            +-- 불린값(boolean)
            +-- undefined
            +-- null
      +-- 참조 타입
            +-- 객체 (Object)
                +-- 배열 (Array)
                +-- 함수 (Function)
                +-- 정규표현식
```
